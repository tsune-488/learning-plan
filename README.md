# 学習計画表アプリ

## 概要
東京都の公立中学校を3校経験する中で、学年が変わるたびに  
「学習計画表の書式」について毎回議論が起こっていることに課題を感じていました。

実際には、どの学校・どの学年でも似た書式が使われており、  
同じ「青色」の画用紙に学習計画表を貼り付けて運用している状況でした。

そのため、**どの学校・どの学年でも共通して使える書式であれば、  
アプリとして統一した方が良い**と考え、  
学習計画表をWebアプリとして開発しました。

生徒1人1台の学習端末環境を想定し、生徒が日々の学習計画・実施内容を入力、  
教員がそれを一括で確認・管理することで、  
**毎日約30分かかっていた確認作業を約5分に短縮できる設計**としています。


---

## 使用技術
- Java 21  
- Spring Boot 3.x  
- Thymeleaf  
- MyBatis  
- MySQL  
- HTML / CSS / JavaScript  
- Gradle  

---

## 機能一覧

### 教員機能
- ログイン
- 学習計画期間（テスト期間）の登録
- 期間学習中の、日付ごとの学習計画・実施内容の確認
- 登録済み生徒名簿の確認

### 生徒機能
- ログイン
- 学習計画・実施内容の入力

---

## 工夫した点
- 教員用・生徒用で画面および機能を明確に分離
- テスト（学習計画期間）単位で生徒が紐づくデータ設計
- ログイン・新規登録時の入力バリデーション実装
- ICTが苦手な教員でも操作できる、シンプルなUI設計
- 教育現場での実運用を想定した画面遷移と操作回数の最小化
